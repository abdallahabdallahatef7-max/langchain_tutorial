## 📌 About This Repository

**LangChain_Course** is a structured, notebook-based learning course that teaches you how to use **LangChain**, a powerful framework for building applications powered by Large Language Models (LLMs).

Instead of just calling an LLM API, LangChain allows you to:
- Chain multiple LLM calls together  
- Add memory and reasoning  
- Connect LLMs with documents, PDFs, and external data  
- Build Retrieval-Augmented Generation (RAG) systems  
- Create production-ready AI workflows  

This repository focuses on **learning by building**, not theory alone.
